# Squish Logs Minecraft
https://squish.wtf/

### Loggers
- System
  - Server Start/Stop
  - Console Command ran
- Players
  - Join/Leave
  - Chat
  - Command
  - Death
  - Teleport
  - Gamemode Change

### Compliling
Run the "shadowJar" gradle job. The jar has to be shadowed so the user doesn't have to worry about also installing the dependencies (aka the websocket library I'm using).
